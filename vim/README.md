# vimfiles

Portable vim configuration

## Install
```
git clone git://github.com/fsproru/vimfiles ~/.vimfiles
cd ~/.vimfiles
./install.sh
```

## Uninstall
```
cd ~/.vimfiles
./uninstall.sh
```
